# Tutorial: LLM agents for chemistry
A tutorial on LLM agents for computational chemistry applications.

This project works with these the **[smolagents](https://github.com/huggingface/smolagents)** framework for designing agents. **[RDKit](https://github.com/rdkit/rdkit)** and **[Morfeus](https://github.com/digital-chemistry-laboratory/morfeus)** are utilized for handling molecular data and calculating descriptors.
